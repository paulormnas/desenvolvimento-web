# Estudo Dirigido 2: Interatividade e APIs com JavaScript

Este estudo dirigido tem como objetivo aprofundar seus conhecimentos em desenvolvimento web, adicionando dinamismo e interatividade às suas páginas através do **JavaScript** e do consumo de **APIs públicas**.

Você desenvolverá uma aplicação web funcional que busca informações externas e as apresenta de forma moderna e atraente.

## 💡 Sugestões de Temas

Para este trabalho, você pode escolher um dos temas abaixo ou propor algo similar:

- **Dashboard de Clima:** Uma página que busca a previsão do tempo para diferentes cidades.
- **Buscador de Filmes:** Um catálogo onde o usuário pesquisa por títulos e vê detalhes como sinopse e avaliação.
- **Conversor de Moedas:** Uma ferramenta que utiliza taxas de câmbio em tempo real para conversão.
- **Catálogo de Personagens:** Uma enciclopédia interativa (Ex: Rick and Morty, Star Wars). Não vale repetir Pokémon que já foi apresentado na sala de aula
- **Notícias de Tecnologia:** Um agregador que exibe as últimas novidades de portais de tecnologia.

## 📋 Critérios de Avaliação

Seu projeto será avaliado com base nos seguintes pontos:

- **HTML Semântico:** Uso correto de tags como `<header>`, `<main>`, `<section>`, `<article>`, `<footer>`, etc.
- **Boas Práticas de CSS:** Organização do código, separação clara entre Classes e IDs, e uso de variáveis CSS se necessário.
- **Design Moderno:** Estética visual moderna, com boa escolha de cores, tipografia e espaçamentos.
- **Consumo de API:** Uso da função `fetch()` para buscar dados reais de uma API pública.
- **Interatividade e Animações:** Uso de transições e animações CSS ou JavaScript para tornar a experiência do usuário mais fluida e dinâmica.

## 🚀 Passo a Passo

### 1. Organização do Projeto

Crie uma pasta chamada `projeto-ed2` e organize seus arquivos:

- `index.html` (Estrutura)
- `style.css` (Estilização)
- `script.js` (Lógica e APIs)
- Pasta `assets/` (Para imagens, ícones ou fontes locais)

### 2. Escolha da API

Existem diversas APIs gratuitas que você pode utilizar (algumas podem exigir uma chave gratuita - API Key):

- [OpenWeatherMap](https://openweathermap.org/api) (Clima)
- [The Movie Database (TMDB)](https://www.themoviedb.org/documentation/api) (Filmes)
- [ExchangeRate-API](https://www.exchangerate-api.com/) (Moedas)
- [JSONPlaceholder](https://jsonplaceholder.typicode.com/) (Dados genéricos para testes)

### 3. Implementação

1. **Estrutura:** Foque primeiro em um HTML limpo e acessível.
2. **Estilo:** Aplique um design moderno. Não pense em "Dark Mode" neste momento, opte por "Glassmorphism" ou layouts minimalistas.
3. **Lógica:** Implemente o JavaScript para capturar eventos do usuário (como cliques ou digitação), buscar os dados na API e atualizar o conteúdo da página dinamicamente (Manipulação do DOM).
4. **Polimento:** Adicione animações de entrada para os elementos e efeitos de hover nos botões e cards.

### 4. Publicação no GitHub

1. Crie um novo repositório público no seu [GitHub](https://github.com/).
2. Faça o upload dos arquivos.
3. (Opcional) Ative o **GitHub Pages** para que possamos ver o site funcionando online.
