# Estudo Dirigido 1: Criação de Site Institucional

Este estudo dirigido tem como objetivo guiá-lo na criação do seu primeiro site utilizando as tecnologias fundamentais da web: **HTML5** e **CSS3**.

Você desenvolverá um site de apresentação para uma empresa fictícia. O projeto deve ser simples, porém bem estruturado e visualmente agradável.

## 📋 Requisitos do Projeto

O site deve conter obrigatoriamente as seguintes seções/páginas:

1. **Home (Página Inicial):** Apresentação geral da empresa, missão ou destaque principal.
2. **Sobre:** História da empresa, valores ou equipe.
3. **Contato:** Página para interação com o usuário. Deve conter obrigatoriamente um formulário HTML (`<form>`) com:
    - Pelo menos 3 caixas de texto (ou `textarea`) para: **Nome**, **E-mail** e **Mensagem**.
    - Um **botão de envio** (`<button>` ou `<input type="submit">`).
    - **Comportamento:** Ao clicar no botão de envio, o formulário deve redirecionar o usuário de volta para a página inicial (`index.html`). Dica: use o atributo `action` da tag `<form>`.

## 🚀 Passo a Passo

### 1. Organização do Projeto

Crie uma pasta no seu computador chamada `projeto-ed1`. Dentro dela, organize seus arquivos:

- `index.html` (Página Home)
- `sobre.html` (Página Sobre)
- `contato.html` (Página Contato)
- `style.css` (Arquivo de estilos)
- Pasta `img/` (Para suas imagens e logotipo)

### 2. Estrutura HTML

Cada página deve possuir uma estrutura semântica básica:

- `<header>`: Com o logotipo e o menu de navegação (`<nav>`).
- `<main>`: Conteúdo principal da página.
- `<footer>`: Rodapé com direitos autorais e links úteis.

> [!IMPORTANT]
> Certifique-se de que os links no menu de navegação funcionem corretamente em todas as páginas, permitindo que o usuário navegue entre Home, Sobre e Contato.

### 3. Estilização com CSS

Utilize o arquivo `style.css` para:

- Definir uma paleta de cores harmoniosa.
- Escolher tipografias legíveis (recomenda-se o uso de Google Fonts).
- Aplicar margens, preenchimentos (padding) e alinhar os elementos.
- Criar um layout responsivo (que se ajuste bem a diferentes tamanhos de tela).

### 4. Publicação no GitHub

1. Crie um novo repositório no seu [GitHub](https://github.com/).
2. Faça o upload de todos os arquivos do seu projeto.
3. Certifique-se de que o repositório seja **público**.
