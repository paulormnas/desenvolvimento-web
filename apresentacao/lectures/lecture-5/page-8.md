# Tailwind CSS: Prós e Contras

Apesar de gerar certa polêmica na comunidade por adicionar muitas classes direto no HTML, o Tailwind é atualmente uma das bibliotecas CSS mais utilizadas e amadas pelas suas enormes vantagens:

**Prós**

* **Criatividade sem limites rígidos de temas:** Você não fica preso em componentes genéricos com "cara de Bootstrap". É possível criar um layout completo sem abrir arquivos CSS externos.
* Componentização em bibliotecas JavaScript modernas (como React) criam fluxos visuais poderosos em segundos.

**Contras**

* **O HTML pode ficar poluído visualmente:** Blocos podem ter dezenas de classes no próprio atributo (ex: `class="flex flex-col space-y-2 mt-4 max-w-lg lg:w-[40%] text-sm etc..."`).
* **Curva de aprendizado:** Você precisa se acostumar com a nomenclatura única das classes utilitárias.
