# Frameworks CSS

Escrever as propriedades do zero para um botão (padding, cor, tamanho da letra, borda arredondada) milhares de vezes começa a esgotar o desenvolvedor. Para isso serve o **ecossistema de Frameworks CSS**, que são bibliotecas com regras visuais criadas previamente e prontas para uso.

Frameworks de base para "Componentes Prontos":

* **Bootstrap:** Um dos mais influentes e famosos do mundo, com foco no sistema de grid de 12 colunas e componentes pré-estilizados. Exemplo: Você simplesmente aplica `<button class="btn btn-primary">` e um botão azul padronizado aparece.
* **Bulma:** Utiliza muito grid/flexbox e classes fáceis
* **Materialize / Material-UI (React):** Define designs influências na política e ecossistema UI e UX de animações e layout imposto pelo proprio *Google* da vida moderna no Android.
