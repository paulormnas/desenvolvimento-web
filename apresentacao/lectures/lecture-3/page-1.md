# 1. Introdução ao CSS

O CSS (Cascading Style Sheets) dita a **apresentação e estilo** das páginas web, separando o design da estrutura (HTML).

- **Estrutura vs Apresentação:** O HTML cria o "esqueleto" e o CSS adiciona a "pele" e as "roupas".
- **Sintaxe Básica:** Uma regra CSS possui um *seletor* e um bloco de *declarações* (propriedades e valores).

```css
seletor {
  propriedade: valor;
}

h1 {
  color: blue;
  font-size: 24px;
}
```
