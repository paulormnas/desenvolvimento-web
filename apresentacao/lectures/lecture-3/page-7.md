# 4. Cores e Fundos (Backgrounds)

O CSS permite controlar as cores de texto e de fundo de forma muito flexível.

- **Sistemas de Cores:**
  - Nomes predefinidos: `red`, `blue`, `tomato`
  - Hexadecimal: `#ff0000`, `#000`
  - RGB / RGBA: `rgb(255, 0, 0)`, `rgba(0, 0, 0, 0.5)` *(Aplica transparência Alfa)*
  - HSL / HSLA: `hsl(120, 100%, 50%)`

### Estilização de Fundo (`background`)

- `background-color`: Define cor sólida.
- `background-image`: Usa uma imagem (`url('imagem.png')`) ou gradiente.
- `background-size`: Controla o tamanho (`cover`, `contain`).
- **Gradientes:** `linear-gradient(to right, red, blue)` e `radial-gradient()`.
