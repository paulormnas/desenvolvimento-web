# 5. Tipografia e Formatação de Texto

<!-- Garantindo que a leitura seja agradável e combinada com o design. -->

- **Fontes base:**
  - `font-family`: `Arial, sans-serif` (sempre inclua uma fonte genérica para fallback).
  - É essencial usar Web Fonts (ex: Google Fonts) via `@import` ou `<link>`.
- **Propriedades comuns:**
  - `font-size`: Tamanho da fonte (usar pixels `px`, relativas `em` ou `rem`).
  - `font-weight`: Espessura (`normal`, `bold`, ou números como `400`, `700`).
  - `font-style`: `normal` ou `italic`.

- **Formatação do texto:**
  - `text-align`: `left`, `center`, `right`, `justify`.
  - `text-decoration`: Útil para remover sublinhado de links `none` ou adicionar `underline`.
  - `line-height`: Aumenta o espaçamento das linhas para melhorar a leitura.
