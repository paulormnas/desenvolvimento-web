# 14. Conclusão e Próximos Passos

Hoje vimos o básico necessário para tornar as páginas web interativas.

- Sintaxe básica e variáveis.
- Lógica de programação (condicionais e loops).
- Manipulação do DOM para mudar o que o usuário vê.
- Requisições assíncronas com Fetch.

### 🚀 Desafio Prático:
Tente selecionar um botão do seu HTML, adicione um clique, busque um dado em uma API pública (como a PokéAPI ou JSONPlaceholder) e exiba esse dado em um parágrafo da tela.
