# 12. Rodando e Debugando no Navegador

Todo navegador moderno tem ferramentas incríveis para desenvolvedores (F12 ou Ctrl+Shift+I).

- **Console**: Para testar pequenos trechos de código e ver logs (`console.log`).
- **Sources (Fontes)**: Onde a mágica acontece.
  - **Breakpoints**: Clique na linha de código para pausar a execução.
  - **Watch**: Monitore o valor de variáveis em tempo real enquanto o código roda.
  - **Step Over/Into**: Caminhe linha a linha para entender a lógica.

> "Se você não está usando o Debugger das DevTools, você está programando no escuro."
