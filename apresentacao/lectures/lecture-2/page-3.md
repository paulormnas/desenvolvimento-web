# O que é HTML Semântico? 🏗️

<div>
A partir de agora, vamos organizar nosso <code>&lt;body&gt;</code> usando <strong>HTML Semântico</strong>.
<br>
</div>

<v-clicks>

**Por que usar semântica?**

1. **Acessibilidade:** Leitores de tela para pessoas com deficiência visual entendem melhor a estrutura da página.
2. **SEO (Otimização para Motores de Busca):** O Google entende melhor do que se trata o seu site se você usar as tags corretas.
3. **Manutenção:** Seu código fica muito mais fácil de ler e entender por você ou por outros programadores.

Em vez de usar apenas `<divs>` (uma caixa genérica), usamos tags com **significado**.

</v-clicks>

<div class="text-center mt-4 text-xl">
