# Dividindo Assuntos: `<section>` 📦

<div>
A tag <code>&lt;section&gt;</code> serve para dividir uma página longa em <strong>seções</strong>. Pense nela como capítulos de um livro ou partes de um artigo grande.
<br>
</div>

<v-clicks>

Use `<section>` sempre que houver um grupo de conteúdo relacionado. Idealmente, toda `<section>` deve ter um título (como um `<h2>` ou `<h3>`).

</v-clicks>

<v-click>

```html
<section>
    <h2>O que levar na mochila?</h2>
    <p>Dicas para arrumar a mochila antes de uma trilha...</p>
    <ul>
        <li>Água</li>
        <li>Protetor Solar</li>
        <li>Lanches</li>
    </ul>
</section>
```

</v-click>
