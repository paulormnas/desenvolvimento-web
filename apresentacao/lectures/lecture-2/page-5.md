# Navegação do Site: `<nav>` 🧭

<div>
A tag <code>&lt;nav&gt;</code> é usada para envolver blocos de <strong>links de navegação importantes</strong>, geralmente o menu principal do seu site.
<br>
</div>

<v-clicks>

Ela ajuda os motores de busca e leitores de tela a encontrarem mais rápido como o usuário se move pelo seu site. O `<nav>` geralmente vai dentro do `<header>`, mas também pode ficar sozinho na página para um rodapé.

</v-clicks>

<v-click>

```html
<nav>
    <ul>
        <li><a href="inicio.html">Início</a></li>
        <li><a href="produtos.html">Nossos Produtos</a></li>
        <li><a href="contato.html">Fale Conosco</a></li>
    </ul>
</nav>
```

</v-click>
