# Informações Extras: `<aside>` 📎

<div>
A tag <code>&lt;aside&gt;</code> representa conteúdo que tem relação com a página principal (ou com o artigo onde está inserido), mas é <strong>"tangencial" ou complementar</strong>.
<br>
</div>

<v-clicks>

O exemplo mais comum para o uso desta tag é criar uma barra lateral (sidebar) com links relacionados, anúncios, uma mini biografia do autor, ou então mostrar os posts mais lidos do mês.

</v-clicks>

<v-click>

```html
<main>
    <article>
        <h2>Receita de Bolo de Cenoura</h2>
        <p>Esta receita é deliciosa...</p>
    </article>

    <aside>
        <h3>Veja Também:</h3>
        <ul>
            <li>Receita de Bolo de Fubá</li>
            <li>Cobertura de Chocolate Simples</li>
        </ul>
    </aside>
</main>
```

</v-click>
