# 📘 Lista de Exercícios de HTML

### Orientação

Para cada exercício crie um novo arquivo HTML. Evite usar um único arquivo para diferentes exercícios desta forma é mais simples acompanhar as mudanças e não confundir os resultados de um exercício com o próximo.

# Exercícios
## 1. Criar Estrutura Básica de Página

Crie um arquivo `index.html` contendo a estrutura mínima: `<!DOCTYPE html>`, `<html>`, `<head>`, `<body>`.  
Dentro do `<body>`, escreva um texto apresentando seu nome usando um `<h1>`.

---

## 2. Lista de Interesses

Crie uma lista ordenada (`<ol>`) com **três interesses pessoais**.  
Abaixo, crie uma lista não ordenada (`<ul>`) com **três comidas favoritas**.  
Use `<li>` para os itens.

---

## 3. Inserir Imagem e Legenda

Insira uma imagem usando `<img>`.  
Abaixo dela, escreva uma legenda usando `<p>` e estilize parte do texto com `<span>`.

---

## 4. Links Internos e Externos

Adicione um link externo para um site que você gosta usando `<a>`.  
Crie também um link interno que leve a outra seção da mesma página (use `id`). Para isso pesquise como utilizar id em outras tags como `<div>`, `<span>` e etc.

---

## 5. Criar uma Seção com Títulos Hierárquicos

Crie uma seção explicando seu hobby favorito usando `<h2>`, `<h3>` e `<p>`.  
Mostre como os títulos se organizam em hierarquia.

---

## 6. Pesquisar e Usar 3 Tags Novas

Pesquise na internet **três tags HTML que você ainda não conhece**.  
Crie uma pequena demonstração de cada tag e explique com um texto breve para que ela serve.

---

## 7. Criar um Mini Portfólio

Monte uma página simples contendo:

- Um título `<h1>`
    
- Uma foto sua ou avatar `<img>`
    
- Uma lista de habilidades (`<ul>`)
    
- Um link para um projeto (real ou fictício) `<a>`
    

---

## 8. Primeira Página Estilizada (Sem CSS!)

Usando apenas tags HTML, tente mudar a aparência do texto com tags como `<strong>`, `<em>`, `<mark>`, `<small>` etc.  
Pesquise pelo menos duas tags que alterem estilo de fonte.

---

## 9. Criar um Menu Simples

Monte um menu de navegação usando `<nav>` (pesquise sobre esta tag).  
Dentro dele, adicione links `<a>` que leve a outras páginas como google.com.br ou g1.globo.com.

---

## 10. Criar um Artigo Simples

Crie uma página com estrutura de artigo contendo:

- `<header>`
- `<section>`
- `<article>`
- `<footer>`

Pesquise essas tags e use-as para organizar o conteúdo da forma mais semântica possível.
