# CSS

## Cascading Style Sheets

<div class="mt-8 text-lg">

<img src="/lectures/lecture-0/css3.svg" class="h-16 mb-4" />

O CSS define o **estilo** e a **aparência** da página.

Ele diz ao navegador *como* as tags do HTML devem ser mostradas (cor, tamanho, fonte, posição).

Conseguimos selecionar elementos e aplicar regras de estilo:

- `color: red;`
- `background-color: blue;`
- `font-size: 20px;`

</div>

::right::

<div class="ml-8 mt-16 p-6 rounded-xl bg-gray-900 shadow-2xl">

```css
/* Exemplo de CSS */

h1 {
  color: blue;        /* Texto azul */
  text-align: center; /* Centralizado */
}

p {
  font-family: Arial; /* Fonte Arial */
  font-size: 18px;    /* Tamanho 18px */
}

button {
  background-color: green; /* Fundo verde */
  color: white;            /* Texto branco */
  border-radius: 5px;      /* Bordas arredondadas */
}
```

</div>
